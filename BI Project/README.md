# Student Performance BI Project

This repository contains a Business Intelligence project focused on analyzing student academic performance.

## Project Goal
To identify patterns related to student success and academic risk using data-driven analysis.

## Tools & Technologies
- ETL: n8n
- Data Model: Star Schema (Fact & Dimensions)
- Visualization: BI Dashboards
- Data Source: Student academic data

## Key Insights
- Attendance is strongly associated with academic performance
- Midterm results act as early indicators of risk
- Performance varies significantly across departments

## Business Value
- Early identification of at-risk students
- Targeted academic interventions
- Improved data-driven decision making
